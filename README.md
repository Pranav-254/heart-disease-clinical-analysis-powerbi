# Heart Disease Clinical Data Analysis: Power BI Report

This repository contains a comprehensive Power BI dashboard designed to analyze clinical factors associated with heart disease. The project transforms medical dataset parameters into visual insights to help identify risk factors and patient demographics most affected by cardiac conditions.

## Project Overview
The dashboard provides a diagnostic view of patient health metrics, focusing on the correlation between clinical indicators—such as cholesterol levels, heart rate, and blood pressure—and the prevalence of heart disease.

## Dataset Overview
The analysis is built upon clinical data featuring several key health attributes:
* **Demographics**: Age and Sex.
* **Vital Signs**: Resting Blood Pressure and Maximum Heart Rate.
* **Laboratory Results**: Cholesterol and Fasting Blood Sugar.
* **Electrocardiogram Metrics**: Resting ECG, ST Slope, and Oldpeak.
* **Symptomatic Data**: Chest Pain Type and Exercise-induced Angina.

## Technical Stack
* **Power BI Desktop**: Used for data modeling, DAX calculations, and report authoring.
* **Data Model**: Structured to allow cross-filtering between clinical symptoms and disease outcomes.

## Usage
1. Open the `heart-disease-clinical-analysis-powerbi` file using Power BI Desktop.
2. Use the interactive slicers to filter data by Age, Gender, stc to observe specific patient outcomes.
